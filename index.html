Course Overview

This workshop is a practical introduction to building websites, covering the technical basics of HTML and CSS, with a quick foray into JavaScript and programming concepts in Week Four. Skills learned each week are practiced in one-off assignments and contextualized within a larger personal project built over the course of the workshop. This larger project is intended to serve as a springboard for future coding endeavors.

Course Objectives

By the successful completion of this workshop, students will have:

  • Working knowledge of HTML and CSS syntax
  • An understanding of the differences between and uses of HTML/CSS & JavaScript in the browser
  • Awareness of the structure of web pages and how it affects design decisions
  • Increased ability to find answers online to coding problems
  • And hopefully, excitement for the possibilities of code as a medium, and the desire to continue study.

Class Format

Class time will be divided between lectures, demos, in-class exercises, and troubleshooting.

Lectures will largely be technical in nature, with the goal of introducing students to the terminologies and affordances of the code concepts relevant for upcoming assignments.

Assignments

There are 7 assignments total in this workshop. Assignments will be a combination of prescriptive coding exercises, looser design applications, and straight up online lessons. Assignment 07 is a larger personal project that will be developed over the course of the workshop and serves as the "final."

Some assignments (or parts of assignments) are optional. Assignments that are entirely optional are grayed out in the schedule, while assignments that are partially optional will have those parts grayed out under the assignment description.

Submission of Work

All assignments that yield a website will be published online through Github and reviewed in Chrome. Links will be submitted through Slack in the #_work channel.

Grading

Please arrive on time and prepared for each class. One unexcused absence will result in failure of the course. I value clear communication regarding any circumstances that require an absence or an extension, as early as possible.

Due to the technical nature of this workshop and the (somewhat) steep learning curve, grading of assignments will rely on demonstrated effort rather than final polish. "Effort" will be understood through risktaking in non-prescriptive assignments and documentation (via comments) of code that was attempted but did not work.

Aside from assignments, grading will consider the contribution of comments in Slack on fellow students' work, or of references and inspiration to the #_references channel.

Complete parts 1.1–1.6 of Learn HTML, and parts 1, 2, 4, and 5 of Learn CSS.

Learn HTML 1: Elements and Structure

  • 1.1 Introduction to HTML
  • 1.2 HTML Anatomy
  • 1.3 The Body
  • 1.4 HTML Structure
  • 1.5 Headings
  • 1.6 Divs

Learn CSS 1: Selectors and Visual Rules (all)

Learn CSS 2: The Box Model (all)

Learn CSS 4: Color (all)

Learn CSS 5: Typography (all)

02 Styling Text

Finish styling the your HTML text from the in-class exercise. Consider changing the typeface or colors. Upon completing Assignment 01, consider adding width, margins, or padding in CSS and/or revisiting your HTML to wrap content in divs.

03 Codecademy CSS

Complete parts 3, 6, and 7 of Learn CSS.

Display and Positioning (all)

Learn CSS 6: Layout with Flexbox (all)

Learn CSS 7: Grid (all)

04 CSS Layout Games

Review CSS layout properties with the following:

  • Flexbox Froggy
  • CSS Grid

This assignment is optional.

05 Responsive Design

Finish marking up the demo we began in class, and make it responsive using media queries.

06 CSS Animation: Marquee

Use CSS Animations to make two marquees. One marquee should run constantly in a fixed header or footer. The other marquee should only run when a user hovers over it.

This assignment is optional.

07 Personal Project

Create a single-page website using your choice of content. This project should demonstrate a cumulation of the skills we've learned in this workshop.

This assignment can be treated as the start of a larger web project, or it can exist on its own. The important thing is that you feel invested in its outcome. We will be working on this assignment over the course of the workshop, and a recommended week-to-week workflow is specified in the schedule.

If you are having trouble deciding on a project idea, you could follow this prompt:

  • Pick an expressive piece of text, whether it's song lyrics, a poem, a manifesto, etc. Combine this text with imagery. Consider how web technologies can be an expressive platform for text with affordances beyond the printed page.

Due 1 week after the last class meeting.
